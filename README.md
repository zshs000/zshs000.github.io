# zshs000.github.io

个人经历展示网站，使用 [Astro](https://astro.build) 搭建，Masonry Flow 设计风格。

## 本地开发

```bash
npm install
npm run dev
```

访问 `http://localhost:4323`

## 构建

```bash
npm run build
```

## 部署

推送到 `master` 分支后，GitHub Actions 自动构建并部署到 GitHub Pages。

## 技术栈

- Astro
- 纯 CSS（组件作用域）
- 原生 JS（滚动动画、打字机效果）
- GitHub Pages + GitHub Actions
